## 🎯 Descripción

SmartHotel es un sistema completo de gestión hotelera que integra **React** (panel web), **Flutter** (app del huésped), **Express + MongoDB** (backend), y **RabbitMQ** (procesamiento asíncrono). Demuestra capacidad de construir un producto con múltiples interfaces y servicios coordinados.

---

## ✨ Features

### Panel web (React)
- [ ] Dashboard de ocupación del hotel
- [ ] Gestión de habitaciones y servicios
- [ ] Vista de solicitudes de huéspedes en tiempo real
- [ ] Reportes de feedback y satisfacción

### App del huésped (Flutter)
- [ ] Check-in digital
- [ ] Solicitud de servicios
- [ ] Control de TV de la habitación
- [ ] Chat con recepción

### Backend (Express + MongoDB)
- [ ] API unificada para web y mobile
- [ ] Eventos con RabbitMQ (nueva solicitud → notificar staff)
- [ ] WebSockets para comunicación en tiempo real
- [ ] Autenticación diferenciada (staff vs huésped)

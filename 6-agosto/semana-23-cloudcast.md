## 🎯 Descripción

CloudCast es el proyecto capstone principal: una **plataforma de TV en la nube a escala reducida** que integra todo lo aprendido. Incluye: panel de administración (React), backend de gestión de contenido (Express + MongoDB), sistema de señales simulado, y despliegue con Docker y Kubernetes. Es un **mini-Yuvod funcional**.

---

## ✨ Features

### Panel de administración (React)
- [ ] Dashboard con KPIs del sistema
- [ ] Gestión de canales y contenido VOD
- [ ] Gestión de hoteles y habitaciones
- [ ] Monitoreo de señales en tiempo real
- [ ] Gestión de usuarios y permisos
- [ ] Reportes y analíticas

### Backend (Express + MongoDB)
- [ ] API REST completa con autenticación
- [ ] Microservicios: auth, content, signals, analytics
- [ ] Comunicación entre servicios con RabbitMQ
- [ ] WebSockets para tiempo real
- [ ] MongoDB con índices optimizados

### Infraestructura
- [ ] Docker Compose para desarrollo local
- [ ] Kubernetes manifests para producción
- [ ] CI/CD con GitHub Actions
- [ ] Health checks y monitoring

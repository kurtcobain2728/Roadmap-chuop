# ☁️ Junio — DevOps, Docker, Kubernetes & Microservicios

> _"El código que no se puede desplegar es código incompleto."_

## 🎯 Objetivo del mes

Dominar infraestructura, containerización, orquestación y arquitecturas distribuidas. Este mes cubre **Docker, Kubernetes, RabbitMQ, CI/CD y microservicios** — requisitos clave de Yuvod.

## 🧠 Habilidades que se desarrollan

- Docker multi-stage builds y Docker Compose avanzado
- Kubernetes: pods, deployments, services, ingress
- Microservicios con comunicación inter-servicio
- Colas de mensajes con RabbitMQ
- CI/CD con GitHub Actions
- Helm charts para Kubernetes
- Monitoring y health checks

---

## 🔗 Cómo se conectan los proyectos

```
Semana 14: ShipIt
    │  Aprende: GitHub Actions, Docker → pipeline básico CI/CD
    ▼
Semana 15: MicroServices
    │  Aprende: Múltiples Express services, Docker network → inter-comunicación
    ▼
Semana 16: QueueFlow
    │  Aprende: RabbitMQ, pub/sub, workers → procesamiento asíncrono
    ▼
Semana 17: K8sDeploy
    │  Aprende: K8s pods, deployments, services → orquestación
    ▼
Semana 18: CICDPipeline
       Aprende: Pipeline completo → CI/CD profesional con testing en K8s
```

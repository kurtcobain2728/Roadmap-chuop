# 🔗 Mayo — Full Stack Integration

> _"El verdadero poder está en conectar frontend y backend en experiencias completas."_

## 🎯 Objetivo del mes

Construir aplicaciones **full-stack completas** que integran React (frontend) con Node.js/Express/MongoDB (backend). Cada proyecto entrega una experiencia de usuario de punta a punta, demostrando capacidad para trabajar en ambos lados del stack simultáneamente.

## 🧠 Habilidades que se desarrollan

- Desarrollo full-stack (React + Express + MongoDB)
- Integración frontend ↔ backend completa
- WebSockets y tiempo real
- Diseño de experiencias de usuario fluidas
- Gestión de estado con React Query y Zustand
- API Gateway y patrones de arquitectura
- PostgreSQL como tecnología adicional valorable

---

## 🔗 Cómo se conectan los proyectos

```
Semana 09: GuestHub
    │  Aprende: React + Express + MongoDB → full-stack básico del dominio hotelero
    ▼
Semana 10: StreamManager
    │  Aprende: WebSockets, control de streams → tiempo real bidireccional
    ▼
Semana 11: BookingEngine
    │  Aprende: Lógica de negocio compleja, pagos → motor de reservas
    ▼
Semana 12: AnalyticsDash
    │  Aprende: Aggregation, Chart.js → dashboards analíticos
    ▼
Semana 13: APIGateway
       Aprende: Rate limiting, caché, auth → gateway centralizado
```

## 🎯 Descripción

GuestHub es una plataforma full-stack que simula el portal de experiencia del huésped de un hotel: permite ver canales disponibles, acceder a contenido VOD, solicitar servicios, ver información del hotel y dejar feedback. El primer proyecto que integra completamente frontend React con backend Express/MongoDB.

---

## ✨ Features

### Frontend (React + MUI)

- [ ] Portal del huésped con pantalla de bienvenida personalizada
- [ ] Catálogo de canales de TV disponibles
- [ ] Librería de contenido VOD (películas, series)
- [ ] Solicitud de servicios (room service, limpieza, etc.)
- [ ] Información del hotel (mapa, restaurantes, amenidades)
- [ ] Formulario de feedback/encuesta de satisfacción
- [ ] Autenticación por número de habitación + apellido
- [ ] Responsive (tablet-first — como una tablet en la habitación)
- [ ] Dark/Light mode

### Backend (Express + MongoDB)

- [ ] API REST para todos los recursos del huésped
- [ ] Autenticación de huésped (simplificada)
- [ ] CRUD de solicitudes de servicio
- [ ] Gestión de feedback y encuestas
- [ ] Endpoints de contenido (canales, VOD)
- [ ] Logging de actividad del huésped

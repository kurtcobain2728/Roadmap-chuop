# 🏗️ Marzo — Backend con Node.js, Express & MongoDB

> _"Todo gran sistema empieza con una API bien diseñada."_

## 🎯 Objetivo del mes

Establecer las bases sólidas del backend construyendo APIs RESTful profesionales con **Node.js, Express y MongoDB** — un stack moderno y demandado. Cada proyecto demuestra buenas prácticas: TypeScript estricto, tests, Docker, documentación y arquitectura limpia.

## 🧠 Habilidades que se desarrollan

- Diseño de APIs RESTful con Express y TypeScript
- Modelado de datos con MongoDB y Mongoose
- Autenticación y autorización (JWT, roles, permisos)
- Búsqueda avanzada con Elasticsearch
- Validación con Zod
- Containerización con Docker
- Testing con Vitest/Jest + Supertest
- Documentación de APIs (OpenAPI/Swagger)

---

## 🔗 Cómo se conectan los proyectos

```
Semana 01: HotelAPI
    │  Aprende: Express, MongoDB, Mongoose, Docker, CRUD, tests
    ▼
Semana 02: AuthNode
    │  Aprende: JWT, bcrypt, roles, RBAC → reutilizable en todos los proyectos
    ▼
Semana 03: ContentDB
    │  Aprende: MongoDB avanzado, agregaciones, gestión multimedia → base de contenido
    ▼
Semana 04: SearchService
       Aprende: Elasticsearch, indexación, búsqueda → motor de búsqueda profesional
```
